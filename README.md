OS-Nexus-Studio is a program designed to help you understand how an operating system works by simulating its core components. It shows how files are managed, how multiple programs share the CPU, and how different scheduling methods affect task execution.

The software combines a backend made with C++ to handle processing and a Python graphical user interface (GUI) to display everything you need to see, making the concepts clear and easy to follow.
